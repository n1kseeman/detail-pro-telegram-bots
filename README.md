# Detail Pro — Telegram-бот для детейлинг-центра

Готовая демонстрационная версия из двух Telegram-ботов для детейлинг-центра и автомойки. Боты работают на бесплатном Telegram Bot API, без ИИ и без онлайн-оплаты. Стоимость нигде не показывается: в интерфейсе указано, что её рассчитывает специалист после оценки автомобиля.

## Возможности

- Клиент выбирает одну или несколько услуг, указывает автомобиль, комментарий, до 5 фотографий и телефон через безопасную кнопку Telegram.
- Запись доступна на ближайшие 14 дней с учётом рабочих дней, блокировок и занятых слотов.
- SQLite не допускает двойного бронирования: активный слот защищён уникальным индексом базы данных.
- Клиент видит активные и завершённые записи, может отменить активную запись и принять/отклонить предложенное новое время.
- Отдельный админ-бот получает заявку с фото и кнопками подтверждения, отклонения, переноса и связи с клиентом. Все уведомления клиенту отправляет только клиентский бот.
- Админ-меню: сегодняшние и активные заявки, поиск по номеру/телефону, расписание на дату, рабочие дни, слоты, блокировки, услуги и статистика.

## Быстрый запуск

Рекомендуется Python 3.11–3.13.

```bash
cd /Users/n1kseeman/Documents/mx
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
```

Создайте в [@BotFather](https://t.me/BotFather) **два разных бота**, затем откройте `.env` и заполните:

- `BOT_TOKEN` — токен клиентского бота, который видят посетители;
- `ADMIN_BOT_TOKEN` — токен отдельного админ-бота, куда будут приходить заявки;
- `ADMIN_GROUP_ID` — необязательный числовой ID рабочей группы, например `-1001234567890`. Админ-бот продублирует туда каждую заявку, фото и служебные уведомления;
- `MANAGER_USERNAME` — username менеджера без `@`;
- `MANAGER_PHONE` — необязательный номер для кнопки звонка;
- `CENTER_NAME` — название центра, по умолчанию `Detail Pro`.

Запуск:

```bash
python main.py
```

При первом запуске автоматически создаётся `data/detail_pro.db`, начальный список из девяти услуг, слоты `10:00–18:00` и рабочие дни с понедельника по субботу. Часовой пояс проекта — `Europe/Minsk`.

## Демо-сценарий

1. Откройте бота и отправьте `/start`.
2. Нажмите «Записаться», отметьте услуги, заполните данные и поделитесь контактом.
3. После отправки запись получит статус «Ожидает подтверждения».
4. Откройте отдельный админ-бот и отправьте ему `/start`.
5. Админ-бот получит заявку; подтвердите её, отклоните с причиной или предложите свободное время.
6. В админ-меню можно быстро настроить график, слоты и заблокировать день/отдельное время.

### Заявки в рабочую группу

Добавьте админ-бота в нужную группу, отправьте в ней любое сообщение и определите её числовой ID (обычно начинается с `-100`). Внесите его в `ADMIN_GROUP_ID` и перезапустите проект. Любой участник группы сможет пользоваться кнопками обработки заявок, поэтому используйте закрытую рабочую группу.
